## Advent of Code 2020

https://adventofcode.com/2020

## Install dependencies

```bash
npm install
```

### Usage

```bash
# Run a specific day challenge. Eg: Day 3 part 1
npm start 3-1

# Run in debug mode
npm run start:debug 3-1

# Run tests
npm test

# Run tests in debug mode
npm run test:debug
```

### Debug mode

Will enable `node inspect` on port 9999
